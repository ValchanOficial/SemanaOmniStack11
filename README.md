# Projeto Be The Hero - Rocketseat

- Tecnologias
- - Nodejs + Express;
- - SQLite + Knex.js;
- - React;
- - React Native;

- Instale as dependências em todas as pastas
```javascript
    npm install
```
- Cria tabelas
```javascript
    npx knex migrate:latest
```
- Execute a aplicação utilizando o comando
```javascript
    npm start
```
